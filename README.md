## 簡單的網頁抽獎小遊戲

**使用技術:**
  
* Web-Layout:
  * html: 基本排版
  * CSS: 運用@for做出扇形的 rotate，使程式碼更為精簡
  * CSS: 善用:nth-child 呈現出轉盤交錯的顏色

* JavaScript:
  * 利用 Vue.js 框架開發
  * 隨機機率，並動態轉動指針，產生轉盤效果
